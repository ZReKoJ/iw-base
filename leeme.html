Robotron 2000

Descripción

  • Robotron 2000 es un juego para el navegador, en el que programas el comportamiento de un robot para competir contra robots programados por otros usuarios.
  • Falta concretar las mecánicas del juego, y el cómo se debe estructurar el código de los robots.
  • Diseña, edita y guarda los códigos de tus robots.
  • Diseña, edita y guarda tus mapas personalizados.
  • Juega y compite contra los robots de otros usuarios en mapas prediseñados, o en mapas creados por ti o por otros usuarios.
  • Gana partidas y escala posiciones en el ranking.

Vistas

Main Page

http://localhost:8080/
  • Título del juego.
  • Breve descripción del juego.
  • Botones:
    • Play: este botón te permite llegar a la zona de juego, donde se simulará el enfrentamiento usando los códigos de los usuarios.
    • Code Design: te redirige a la página donde se crean y editan los códigos de tus robots.
    • Ranking: muestra la página con la lista de los usuarios con mejores puntuaciones.
    • Map Design: te envía a la página de diseño de mapas.
  • Información: En esta sección aparecen las alertas que avisan a los usuarios de las novedades del juego.

Zona de Juego

http://localhost:8080/playing
  • Título del enfrentamiento. En el aparecen los usuarios que se están enfrentando.
  • Mapa e iconos del juego.
  • Información sobre el juego de ambos jugadores entre los que se incluye:
    • Vida actual del robot.
    • Objetos recogidos.
    • Número de golpes propinados al adversario.
    • Habilidades usadas.
  • Botón que permite llegar a la pantalla de resultados.

Resultados

http://localhost:8080/results
  • Título de ganador en el que aparece el nombre del ganador del enfrentamiento.
  • Información final del juego para ambos jugadores entre los que se incluye:
    • Vida final del robot.
    • Objetos recogidos.
    • Número de golpes propinados al adversario.
    • Habilidades usadas.
  • Botón que redirige a la página principal para poder modificar el código si has perdido o mirar el score si has ganado.

Code Design

http://localhost:8080/code-design
  • Your codes: es una lista con todos los códigos que el usuario ha subido con anterioridad. El elemento que se encuentra en azul es el que se está usando actualmente. Además, el botón “Use this code” lleva a la página principal para permitir al usuario iniciar una nueva partida una vez le ha dicho a su robot qué código es el que debe usar.
  • Upload new code: esta sección permite subir un nuevo código de comportamiento del robot de dos maneras diferentes:
    • From file:
      • Título
      • Input “Name”. Este input se utiliza para establecer un nombre con el que el nuevo código aparecerá en “Your codes”.
      • Input del archivo para seleccionar el archivo deseado desde el almacenamiento local
      • Botón de subida. Este botón mantiene a usuario en la misma página. Lo que hace es recargarla para que aparezca el nuevo código en la lista “Your codes”.
    • Writing:
      • Título
      • Input “Name”. Este input se utiliza para establecer un nombre con el que el nuevo código aparecerá en “Your codes”.
      • Área de texto donde se puede escribir el nuevo código a subir.
      • Botón de subida. Este botón mantiene a usuario en la misma página. Lo que hace es recargarla para que aparezca el nuevo código en la lista “Your codes”.

Map Design

http://localhost:8080/map-design
  • Your maps: es una lista con todos los mapas que el usuario ha creado. El elemento que se encuentra en azul es el que está seleccionado. Además, el botón “Use this map” lleva a la página principal para permitir al usuario iniciar una nueva partida en el mapa seleccionado.
  • Design your map: aquí es donde el usuario podrá editar sus mapas y crear mapas nuevos, está compuesta por los siguientes elementos:
    • Lista lateral con los elementos que se pueden añadir. La idea es utilizar el mecanismo Drag and Drop.
    • Zona de edición del mapa.
    • Botón “Save”. Lleva a la página inicial.
    • Botón “Cancel”. Lleva a la página inicial.

Profile

http://localhost:8080/profile
  • Información básica del usuario. Nombre, rank, y foto de perfil.
  • Número de partidas ganadas, perdidas y empatadas.
  • Opciones desplegables:
    • History: últimas acciones del usuario.
    • Password: .
    • Codes: listado de los códigos de los robots del usuario.
    • Maps: listado de los mapas creados por el usuario.
    • Logout: opción para cerrar sesión.
  • Botón “Back”. Lleva a la página inicial.

Settings

http://localhost:8080/settings
  • Create new map: redirecciona a la vista de Map Design.
  • Create new code: redirecciona a la vista de Code Design.
  • Botones extra.
  • Botón “Back”. Lleva a la página inicial.

Ranking

http://localhost:8080/ranking
  • Lista de los usuarios con mejores puntuaciones ordenados de mayor a menor.
  • Botón “Back”. Lleva a la página inicial.
